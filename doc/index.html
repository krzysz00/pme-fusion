pme_fusion.pl  -- Find fusable loop invariants from partitioned (matrix) expressions
fused_invariants/1When at least one region's past and future fields are unbound, find loop invariants for each operation (represented by a list of regions) in the input, such that the loops can be fused - that is, if the loop invariants for each operation are used to generate an algorithm, the loop bodies of the algorithms can be concatentated without sacrificing the correctness of each invariant.
gen_invariant/1Finds and prints all loop invariants for a given list of tasks.
gen_invariants/1Find and print all fusable loop invariants for the given list of operations, each of which are specified as a list of tasks.
gen_invariants_dedup/1Like gen_invariants/1, but the results are deduplicated before printing.
loop_invariant/1A converience wrapper around fused_invariants/1 that finds all loop invariants for a single operation.
make_pme/2Convenience wrapper around make_pmes/2 for the case when there is one operation.
make_pmes/2Transforms the list of lists of tasks (one task-list per operation being fused) into a list of list of regions, that is, a list of partitioned matrix expressions (PMEs).
make_region/5The lowest-level constructor for a region.
print_invariant/1Print all non-empty regions in the given loop invariant.
print_invariants/1Prints each loop invariant in a list of loop invariants, separated by "then".
print_invariants_sep/1Convenience wrapper around print_invariants/1 that adds a blank line after the invariants.
region_with_tasks/3Convenience wrapper that creates the region Reg with the given Id and Tasks.
test_task_list/2Like gen_invariant/1, but it will report the number of invariants found in NInvariants.
test_task_lists/2Like gen_invariants/1, but it will report the number of invariant series found in NInvariants.
test_task_lists_dedup/2Like gen_invariants_dedup/1, but it will report the number of unique invariant series found in NInvariants.
